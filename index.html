About this lab

This is a static site that generates bookmarklet links for accessibility testing and development. Items marked AI need a model endpoint and key you provide. Many actions are toggles, so click again to revert.

Some sites use Content Security Policy which can block bookmarklets. When that happens, try another page, open DevTools, or run the code from the console.
